## Introduction 
A CIS-4379 class project for an eco-friendly, thought-conscious pet store. This project intends to showcase our combined understanding of how to design, develop, and produce a functional website complete with front-end, backend, and database capabilities. This GitHub project is intended as part of a larger product, complete with a live presentation, presentation slides, and documentation to support the software engineering skills utilized in this website.
	
## Built With
The project is created with:
* PHPMyAdmin
* Visual Studio Code
* MAMP
* MySQL
